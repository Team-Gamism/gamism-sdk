`Gamism.SDK.Core`와 `Gamism.SDK.Extensions.AspNetCore` 패키지 사용 가이드입니다.

== 설치

[source,xml]
----
<PackageReference Include="Gamism.SDK.Core" Version="x.x.x" />
<PackageReference Include="Gamism.SDK.Extensions.AspNetCore" Version="x.x.x" />
----

== 등록

`Program.cs`에 두 줄만 추가하면 모든 기능이 활성화됩니다. 각 기능은 기본적으로 활성화(`Enabled = true`)되어 있습니다.

[source,csharp]
----
builder.Services.AddGamismSdk(options =>
{
    options.ConfigureSwagger(swagger =>
    {
        swagger.Title = "My API";
        swagger.PathsToMatch = ["/api/**"];
    });

    options.ConfigureLogging(logging =>
    {
        logging.NotLoggingUrls = ["/swagger/**", "/health"];
    });
});

app.UseGamismSdk();
----

== CommonApiResponse<T>

서버와 클라이언트가 공유하는 공통 API 응답 포맷입니다.

=== 응답 구조

[source,json]
----
{
  "status": "OK",
  "code": 200,
  "message": "성공",
  "data": <페이로드>
}
----

=== 팩토리 메서드

[cols="2,1,2", options="header"]
|===
| 메서드 | HTTP Status | 설명

| `CommonApiResponse.Success(message)`
| 200 OK
| data 없음

| `CommonApiResponse.Success(message, data)`
| 200 OK
| data 포함

| `CommonApiResponse.Created(message)`
| 201 Created
| data 없음

| `CommonApiResponse.Created(message, data)`
| 201 Created
| data 포함

| `CommonApiResponse.Error(message, status)`
| 지정한 status
| data 없음
|===

== 로깅 필터 (ConfigureLogging)

HTTP 요청과 응답을 자동으로 로깅하는 미들웨어(`IMiddleware`)입니다.

=== 로그 포맷

요청:

[source]
----
[Request] GET /api/users?page=1
----

응답:

[source]
----
[Response] 200 (42ms)
----

=== 설정 옵션

[cols="2,1,3", options="header"]
|===
| 옵션 | 기본값 | 설명

| `Enabled`
| `true`
| 로깅 필터 활성화 여부

| `NotLoggingUrls`
| `[]`
| 로깅에서 제외할 URL 패턴 목록. `*`는 경로 구분자를 제외한 모든 문자, `**`는 경로 구분자 포함 모든 문자와 매칭
|===

[source,csharp]
----
options.ConfigureLogging(logging =>
{
    logging.NotLoggingUrls = ["/health", "/swagger/**"];
    // "/swagger/index.html", "/swagger/v1/swagger.json" 등 모두 제외됨
});
----

== 응답 래퍼 (ConfigureResponse)

컨트롤러의 모든 반환값을 `CommonApiResponse<T>` 형식으로 자동 래핑하는 필터(`IAsyncResultFilter`)입니다.

=== 케이스별 동작

[cols="2,1,2", options="header"]
|===
| 반환값 | HTTP Status | 동작

| 일반 객체 / `Ok(value)`
| 200 OK
| `CommonApiResponse.Success("OK", value)` 로 래핑

| `null` / `void`
| 204 No Content
| 빈 응답

| `ICommonApiResponse` 구현체
| 응답 내 `code` 값
| 그대로 통과 (이중 래핑 없음)
|===

=== 컨트롤러 예시

[source,csharp]
----
[ApiController]
public class UserController : ControllerBase
{
    [HttpGet("/users")]
    public List<User> GetUsers()
    {
        // 자동으로 CommonApiResponse<List<User>>로 래핑됨
        return _userService.GetAll();
    }

    [HttpPost("/users")]
    public CommonApiResponse<User> CreateUser(CreateUserRequest request)
    {
        // ICommonApiResponse 구현체는 이중 래핑 없이 그대로 응답
        return CommonApiResponse.Created("생성 완료", _userService.Create(request));
    }

    [HttpDelete("/users/{id}")]
    public void DeleteUser(int id)
    {
        // void 반환 → 204 No Content
        _userService.Delete(id);
    }
}
----

=== 오류 응답 주의사항

`IActionResult`의 오류 반환 메서드(`NotFound()`, `BadRequest()` 등)는 `ObjectResult`가 아니므로 래퍼를 거치지 않아 응답 형식이 깨집니다. 오류 응답은 반드시 `ExpectedException`을 사용하세요.

[source,csharp]
----
// ❌ body 없는 404 반환 — CommonApiResponse 형식이 아님
return NotFound();

// ✅ GlobalExceptionHandler가 CommonApiResponse 형식으로 변환
throw new NotFoundException("유저를 찾을 수 없습니다");
----

=== 설정 옵션

[cols="2,1,3", options="header"]
|===
| 옵션 | 기본값 | 설명

| `Enabled`
| `true`
| 응답 래퍼 활성화 여부

| `NotWrappingUrls`
| `[]`
| 래핑에서 제외할 URL 패턴 목록. `**` 와일드카드 지원
|===

[source,csharp]
----
options.ConfigureResponse(response =>
{
    response.NotWrappingUrls = ["/swagger/**", "/health"];
});
----

== 예외 처리 (ConfigureException)

`ExpectedException`을 던지면 `GlobalExceptionHandler`가 자동으로 처리하여 `CommonApiResponse` 형식으로 응답합니다. 그 외 예외는 500으로 응답하고 스택 트레이스를 `ILogger`에 기록합니다.

=== 기본 사용법

[source,csharp]
----
// 메시지 직접 지정
throw new ExpectedException(HttpStatusCode.NotFound, "유저를 찾을 수 없습니다");

// status의 ToString()을 메시지로 사용 ("NotFound")
throw new ExpectedException(HttpStatusCode.Forbidden);
----

=== 응답 예시

[source,json]
----
{
  "status": "NOT_FOUND",
  "code": 404,
  "message": "유저를 찾을 수 없습니다"
}
----

=== 내장 예외 클래스

자주 쓰는 상태코드는 전용 클래스를 제공합니다.

[cols="2,1", options="header"]
|===
| 클래스 | HTTP Status

| `BadRequestException`
| 400

| `UnauthorizedException`
| 401

| `ForbiddenException`
| 403

| `NotFoundException`
| 404

| `ConflictException`
| 409
|===

[source,csharp]
----
throw new NotFoundException("유저를 찾을 수 없습니다");
throw new ForbiddenException("접근 권한이 없습니다");
----

=== 성능

`ExpectedException`은 스택 트레이스를 수집하지 않아 일반 예외보다 생성 비용이 낮습니다. `GlobalExceptionHandler`도 `ExpectedException`에 대해서는 스택 트레이스를 로그에 남기지 않습니다.

=== 설정 옵션

[cols="2,1,3", options="header"]
|===
| 옵션 | 기본값 | 설명

| `Enabled`
| `true`
| 예외 핸들러 활성화 여부
|===

== Swagger (ConfigureSwagger)

Swashbuckle 기반 Swagger UI를 자동으로 구성합니다. `CommonApiResponseOperationFilter`가 모든 200 응답 스키마를 `CommonApiResponse` 래핑 구조로 재작성합니다. 컨트롤러 반환 타입이 이미 `ICommonApiResponse`를 구현하는 경우 `data` 필드는 스키마에서 제외됩니다.

=== 설정 옵션

[cols="2,1,3", options="header"]
|===
| 옵션 | 기본값 | 설명

| `Enabled`
| `true`
| Swagger 활성화 여부

| `Title`
| `"API Documentation"`
| API 문서 제목

| `Description`
| `"API Documentation"`
| API 문서 설명

| `Version`
| `"v1"`
| API 버전 (Swagger 엔드포인트 경로에 사용)

| `Group`
| `"default"`
| API 그룹명

| `PathsToMatch`
| `["/**"]`
| 문서화할 경로 패턴 목록. `**` 와일드카드 지원
|===

[source,csharp]
----
options.ConfigureSwagger(swagger =>
{
    swagger.Title = "My API";
    swagger.Version = "v1";
    swagger.PathsToMatch = ["/api/**"];
    // Swagger UI: /swagger/v1/swagger.json
});
----

== 전체 설정 레퍼런스

[source,csharp]
----
builder.Services.AddGamismSdk(options =>
{
    options.ConfigureException(exception =>
    {
        exception.Enabled = true;
    });

    options.ConfigureResponse(response =>
    {
        response.Enabled = true;
        response.NotWrappingUrls = ["/swagger/**", "/health"];
    });

    options.ConfigureLogging(logging =>
    {
        logging.Enabled = true;
        logging.NotLoggingUrls = ["/swagger/**", "/health"];
    });

    options.ConfigureSwagger(swagger =>
    {
        swagger.Enabled = true;
        swagger.Title = "My API";
        swagger.Description = "My API Description";
        swagger.Version = "v1";
        swagger.Group = "default";
        swagger.PathsToMatch = ["/api/**"];
    });
});

app.UseGamismSdk();
----
