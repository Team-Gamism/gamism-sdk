게임 서버(ASP.NET Core)와 Unity 클라이언트를 위한 모듈형 SDK입니다.

== 패키지 구성

[cols="2,1,1,2", options="header"]
|===
| 패키지 | 배포 | 프레임워크 | 설명

| `Gamism.SDK.Core`
| NuGet
| netstandard2.0
| 서버·클라이언트 공유 타입

| `Gamism.SDK.Extensions.AspNetCore`
| NuGet
| net8.0+
| ASP.NET Core 기능 모듈

| `Gamism.SDK.Unity`
| UPM
| netstandard2.0
| Unity 통합 레이어
|===
